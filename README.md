# UNISAT UTXO UNLOCKER

Инструмент для автоматического мерджа и отправки UTXO с множества Unisat кошельков.

t.me/alreadydead

## Установка

pip install -r requirements.txt

Требуется Python 3.12+

## Что делает

1. **Check mode** - проверяет все кошельки на наличие UTXO, показывает баланс в BTC и USD
2. **Process mode** - автоматически мерджит несколько UTXO в один и отправляет на указанный адрес

Если на кошельке несколько UTXO:
- Создаёт merge транзакцию на себя
- Ждёт подтверждения
- Отправляет один UTXO (весь анлокнутый BTC) на destination адрес

Если один UTXO (все разлочено):
- Сразу отправляет на destination адрес
